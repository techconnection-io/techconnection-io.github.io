Nos Rendez-Vous

Depuis 2016, les membres de La Tech Connection travaillent pour rassembler l'écosystème mobile en France en organisant événements et meetups tels que FrenchKit et CocoaHeads Paris. À ces derniers s'ajoutent en 2023 d'événéments dédiés aux technologies Flutter et React Native.

FrenchKit

FrenchKit est la principale conférence française dédiée à l'ensemble des technologies Apple, Swift, iOS et macOS.
Flutter Connection

Flutter Connection est la prémière conférence intérnationale made-in-France dédiée à l'écosytème Flutter. Rendez-vous le 2 juin 2023.
React Native Connection

React Native Connection est la prémière conférence intérnationale made-in-France dédiée à l'écosytème React Native. Rendez-vous le 1 juin 2023.
CocoaHeads Paris

CocoaHeads Paris est le premier meetup Français dédié aux technologies Apple. Fondé en 2009 il rassemble chaque mois les développeurs Swift de la région Parisienne.
React Native Paris

React Native Paris est le nouveau rendez-vous régulier des développeurs React Native.
