Nos Rendez-Vous

Depuis 2016, les membres de La Tech Connection travaillent pour rassembler l'écosystème dev en France en organisant événements et meetups tels FrenchKit et CocoaHeads Paris. À ces derniers s'ajoutent en 2023 d'événéments dédiés aux technologies Flutter et React Native et, en 2024, React.

Swift Connection

Swift Connection est la principale conférence française dédiée à l'ensemble des technologies Apple, Swift, iOS et macOS.

Flutter Connection

Flutter Connection est la prémière conférence intérnationale made-in-France dédiée à l'écosytème Flutter. Rendez-vous le 24 avril 2024 pour sa 2e édition.

React Connection

React Connection est la conférence intérnationale made-in-France dédiée à l'écosytème React. Rendez-vous le 22 avril 2024.

React Native Connection

React Native Connection est la prémière conférence intérnationale made-in-France dédiée à l'écosytème React Native. Rendez-vous le 23 avril 2024 pour sa 2e édition.

CocoaHeads Paris

CocoaHeads Paris est le premier meetup Français dédié aux technologies Apple. Fondé en 2009 il rassemble chaque mois les développeurs Swift de la région Parisienne.

React Native Paris

React Native Paris est le nouveau rendez-vous régulier des développeurs React Native.

Nos Trainings

Animées par de véritables experts du domaine, tels que John Sundell, nos formations couvrent les dernières technologies et techniques de développement, notamment SwiftUI et Swift Concurrency.

Que vous soyez débutant ou confirmé, vous trouverez une formation adaptée à votre niveau de compétence. Consultez notre catalogue de formations en ligne ou contactez-nous pour des formations sur mesure pour vous-même ou votre entreprise.
